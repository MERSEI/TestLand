## Порівняння: AI (Claude Code) vs програміст вручну

### Оцінка часу для ручної розробки

| Задача | AI (Claude) | Програміст вручну | Коментар |

| ProjectsBlock (розкривні картки, інфографіка) | ~5 год | ~16-20 год | Складний компонент: expand/collapse, каруселька, графіки |

### Результат

| Метрика | Значення |
|---------|----------|
| Час AI (Claude Code) | ~26.5 год |
| Оцінка для програміста (middle) | ~60-80 год (8-10 робочих днів) |
| **Прискорення** | **~2.5-3x швидше** |
| Кількість компонентів | 6 (InvestorBlock, StatusSwiper, ProjectsBlock, FormModal, PhotoTicker, VerticalCarousel) |
| Файлових операцій | 152 |
| Стек | React + Vite + Tailwind CSS |

### Чому швидше:
- Миттєве генерування JSX/CSS коду з правильним синтаксисом
- Автоматичне читання макету з Figma (MCP інтеграція)
- Паралельне оновлення даних та логіки рендерингу
- Автоматична перевірка білду після кожної зміни
- Швидкі ітерації pixel-perfect доведення
- Git + деплой в одну команду

### Де AI витрачав більше часу:
- Навігація по Figma canvas (zoom, scroll — не оптимально для canvas-based UI)
- Численні ітерації pixel-perfect доведення (потрібно порівнювати скріншоти)
- Контекстні обмеження — 3+ рази потрібно було продовжувати сесію
